# Calcolatrice-Java

Questa è una semplice calcolatrice Java da linea di comando.

## Funzionalità
- Somma
- Sottrazione
- Moltiplicazione
- Divisione
- Potenza
- Radice quadrata
- Percentuale

## Come funziona
All'avvio mostra un menu numerico e richiede all'utente di scegliere l'operazione desiderata.
- Per le operazioni base (`+`, `-`, `*`, `/`) chiede due numeri.
- Per la potenza chiede la base e l'esponente.
- Per la radice quadrata chiede un solo numero.
- Per la percentuale chiede il valore totale e la percentuale da calcolare.

Dopo aver mostrato il risultato, chiede se si vuole continuare con un'altra operazione.

## Uso
1. Compila i file Java:
   `javac Main.java`
2. Esegui il programma:
   `java Main`
3. Inserisci il numero corrispondente all'operazione desiderata.
4. Fornisci i valori richiesti dal programma.
5. Quando viene chiesto "Vuoi continuare? (s/n)", rispondi `s` per continuare o `n` per uscire.
